# Documentación de Rollercoasters

Este documento proporciona una visión general de las características de las montañas rusas americanas. Los datos se han recopilado y analizado para proporcionar una visión detallada de las diversas características de estas atracciones emocionantes.

## Estructura de los Datos

Los datos se presentan en un dataframe que incluye las siguientes columnas:

- **Inversions**: Número de inversiones en la montaña rusa.
- **year_introduced**: Año en que se introdujo la montaña rusa.
- **latitude, longitude**: Coordenadas geográficas de la montaña rusa.
- **speed1_value, speed_mph**: Velocidad de la montaña rusa en diferentes unidades.
- **height_value, height_ft**: Altura de la montaña rusa en diferentes unidades.
- **Inversions_clean**: Número limpio de inversiones en la montaña rusa.
- **Gforce_clean**: Fuerza G experimentada en la montaña rusa.
